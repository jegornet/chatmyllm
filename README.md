# chatmyllm

A native macOS chat application for interacting with LLM models via OpenRouter API.

## Features

- **Model Selection** - Choose from available OpenRouter models for each chat
- **Customizable Interface** - Adjust font, size, and line spacing
- **Localization** - English and Russian interface
- **Native macOS** - Built with SwiftUI for modern macOS
- **Quick Chat** - Start a new chat via system-wide shortcut ⌥Space

## Requirements

- macOS 14.0 or later
- [OpenRouter](https://openrouter.ai/) API key

## Setup

1. Launch the application
2. Enter your OpenRouter API key in Settings
3. Select models to enable
4. Start chatting
